Challenge

The Problem

南海トラフ地震などの大規模災害で、橋のない離島（大崎上島など）が孤立したとき——
物資輸送も、被害把握も、すべてが遅れる。
課題 01 / 孤立リスク
橋のない離島の脆弱性
広島県・大崎上島のような離島は、橋や海底トンネルを持たない。大規模地震や津波で船が出せなくなると、 外部からの支援が完全に遮断される。
課題 02 / ドローン活用の限界
充電インフラが存在しない
ドローンによる物資輸送・空撮は有効な手段だが、 離島上空に充電設備はない。バッテリー切れが即座に運用不能を意味し、 長距離・長時間飛行の実現が困難だった。
チームの問い

「空中でドローンを充電できれば、
離島の空が救急インフラになる。」

地上に充電ステーションを置くのではなく、 電波そのものを「空のコンセント」として機能させる—— そのアイデアがGOROプロジェクトの出発点となった。

Solution Concept

空のコンセント

Core Concept
空のコンセント
Wireless Power in the Sky — Charging Drones Mid-Air

電波を「ただ飛ばす」のではなく、2本の電波を空中で干渉させ、 反射板で収束させることで特定ポイントに電力を集中させる。 ドローンはその「給電ポイント」を通過しながらバッテリーを補充し、 充電ステーションなしに長時間飛行を実現する——それが「空のコンセント」構想だ。

Technology

Technical Architecture

System 01 — 送電
給電ポイント形成
5.7GHz帯のWi-Fiルーター2台が発する電波を空中で干渉させる。 反射板を用いて干渉によって生まれる強電界を給電ポイントへ収束し、 電力伝送可能な領域を空中に形成することを実証。
5.7GHz Wi-Fi × 2台
反射板による電界収束
給電ポイント形成 実証済み
System 02 — 受電
RF-DC 変換モジュール
ドローン側に搭載するRF-DC変換基板を自作。 電波エネルギーを受信し直流電力へ変換してバッテリーへ還元する。 配線不具合により検証は継続中だが、設計・実装は完了している。

System 03 — 計測
ごろごろシステム
電圧の計算・計測を行う独自ソフトウェアシステム「ごろごろシステム」を開発。 給電効率の定量評価やパラメータチューニングに活用し、 ハードウェア実装の検証基盤として機能する。

Process

Challenges & Learnings

障壁 01 / 電波法規制
業務用送信機の開局申請
高出力の実験には総務省への実験局開局申請が必要。 re-KOSEN期間内（約5ヶ月）での許可取得はできなかったが、 申請書類の準備を進め、次フェーズへの基盤を構築した。
障壁 02 / ハードウェア実装
RF-DC基板の配線不具合
受電モジュール製作時に配線不具合が発生。 設計ミスではなく実装精度の問題として切り分け、 修正フローを確立。短サイクル検証・現場最適の方針で対応した。
対応 / ライセンス取得
第一級・第三陸上特殊無線技士
プロジェクト推進を見据え、坂口は第一級陸上特殊無線技士・ 第三陸上特殊無線技士を取得。電波法制度の理解と安全運用の 基礎知識を実務レベルで保持している。
今後の展開
総務省申請 → 高出力実証
現在は総務省への実験局申請準備を進行中。 許可取得後に業務用送信機での高出力実験を実施し、 「空のコンセント」の実用化検証へ移行する計画。
